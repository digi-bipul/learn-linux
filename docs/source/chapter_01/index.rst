=================================
Chapter 1: Foundations of Linux
=================================

.. epigraph::

   *"The most important design issue … is the fact that Linux is supposed
   to be fun."* — Linus Torvalds, 1992

This chapter plants the seeds.  You will learn what Linux actually *is*,
how it came to dominate the computing world, how to choose a distribution
that fits your needs, and — critically — how to open a terminal and begin
talking to the machine.

By the end of this chapter you will be able to:

* Explain the difference between a *kernel*, an *operating system*, and a
  *distribution*.
* Recount the lineage from Unix through GNU to the modern Linux ecosystem.
* Make an informed choice between Debian, Ubuntu, Fedora, Arch, Alpine,
  and other major families.
* Navigate the Linux filesystem tree with confidence.
* Use fundamental shell commands to inspect, create, move, and delete
  files and directories.
